# Note App - AI Extended

Demo project for **OAISYS - 2025** event  
Talk: [Beyond the Cloud: AI that Runs in the Browser](https://theoaisys.com/session/beyond-the-cloud-ai-that-runs-in-the-browser)

A React-based note-taking application that demonstrates browser-native AI capabilities using Chrome's built-in AI features.

## Features

- Create, edit, and delete notes
- AI-powered features: summarize, proofread, rewrite, generate titles/tags
- Local storage using IndexedDB
- Progressive Web App (PWA) support

## Prerequisites

- Node.js (v16 or higher)
- Chrome browser with AI features enabled (chrome://flags)

## Installation & Run

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:5173`

## Build for Production

```bash
npm run build
npm run preview
```
